# AI Product Studio

A high-converting, full-stack Amazon listing agency website. Built with a sleek, conversion-focused frontend and a real-time Node.js/Socket.io backend for lead management.

## Features
* **Modern Frontend:** Cinematic scroll animations, custom cursors, and interactive Before/After sliders.
* **Real-Time Admin Dashboard:** Live lead tracking, conversion metrics, and visitor analytics powered by WebSockets.
* **API Integration:** Form submissions bypass the browser and go straight to a secure Node.js backend.

## Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Node.js, Express.js
* **Real-Time:** Socket.io
* **Database:** Local JSON File Storage
